class PNGpicture

Detailed Description

Structure which represents part of picture. PNG format pictures are divided into chunks. Most important chunks are critical chunks:

  1. IHDR - always first chunk;
  2. IDAT - chunk with data;
  3. PLTE - color table (optional);
  4. IEND - last chunk which mean end of picture;

Chunk scheme representation |LENGTH|CHUNK_TYPE|CHUNK_DATA = optional|CRC|
